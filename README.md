# Test Yummy
## Description

Api pokemon

## Instalacion

```bash
$ npm install
```

## Correr la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Variables de entorno

```bash
# direccion del api pokemon
$ API_URL = https://pokeapi.co/api/v2

# Direccion de la base de datos
$ DB_URL=***.***.*.***

# Nombre de la base de datos
$ DB_NAME=pokemons


# Puerto de la base de datos
$ DB_PORT=27017

```
